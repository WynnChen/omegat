Что собой представляет OmegaT?
===============

OmegaT — это свободная и открытая мультиплатформенная система автоматизированного перевода (или система CAT —Computer Aided Translation), включающая поиск нечётких совпадений, память переводов, поиск по ключевым словам, глоссарии и использование существующих переводов в обновлённых проектах.

Необходимые условия для установки OmegaT
============================================

Для работы OmegaT необходимо, чтобы в вашей системе была установлена среда выполнения Java Runtime Environment (JRE) версии 1.8 или выше.

Чтобы не тратить времени на поиск, скачивание и установку среды выполнения, рекомендуется использовать установочные пакеты OmegaT, включающие JRE.



Установка OmegaT (Windows)
=============================

Необходимо запустить программу установки.



Установка OmegaT (OS X)
==========================

Распакуйте архив OmegaT.zip. Появившаяся папка будет содержать файл документации и приложение OmegaT. Переместите эту папку в папку приложений или в другое подходящее место.



Установка OmegaT (GNU/Linux)
============================

Поместите архив в любую подходящую папку и распакуйте его — программа OmegaT готова к работе.

Кроме того, для более удобного использования вы может установить программу с помощью скрипта установки
linux-install.sh. Для этого откройте окно эмулятора терминала (консоль), перейдите в папку, содержащую OmegaT.jar и linux-install.sh, и запустите скрипт командой
sudo ./linux-install.sh
или
su -c ./linux-install.sh



Установка OmegaT (Solaris, FreeBSD и другие UNIX-системы)
=========================================================

Поместите архив в любую подходящую папку и распакуйте его — программа OmegaT готова к работе.



Использование Java Web Start для установки OmegaT (все платформы)
=================================================================

Если у вас уже установлена среда JRE, то проще всего установить OmegaT с помощью Java Web Start.

Для этого загрузите указанный ниже файл и запустите его:

  https://omegat.sourceforge.net/webstart/OmegaT.jnlp

В результате на ваш компьютер будут установлены компоненты, необходимые для работы OmegaT, и само приложение. Для последующих запусков наличие соединения с Интернетом не обязательно.



Запуск OmegaT (Windows)
=======================

Если во время установки на рабочем столе был создан ярлык, то просто дважды щелкните по нему мышью.

Программа установки может создать ярлыки в меню «Пуск», на рабочем столе и на панели быстрого запуска. Кроме того, можно вручную создать ярлык для запуска, перетащив файл OmegaT.exe в меню «Пуск», на рабочий стол или на панель быстрого запуска.

Если в файловом менеджере (Проводнике Windows) вы видите только файл OmegaT, а не OmegaT.exe, то включите отображение расширений файлов в настройках Проводника Windows.



Запуск OmegaT (OS X)
====================

Дважды щелкните мышью по приложению OmegaT.

Для удобства запуска просто перетащите приложение OmegaT в док или на панель Finder. Кроме того, OmegaT можно запустить из результатов поиска Spotlight.



Запуск OmegaT (GNU/Linux)
=========================

Если вы воспользовались скриптом установки linux-install.sh, то вы можете запустить OmegaT, нажав

    ALT+F2

и введя в появившемся окне:

    omegat

Для более удобного запуска OmegaT можно использовать входящий в пакет установки скрипт на языке Kaptain (omegat.kaptn). Для использования этого скрипта сначала необходимо установить Kaptain. Затем этот Kaptain-скрипт можно запустить, нажав

    ALT+F2

и введя в появившемся окне:

    omegat.kaptn



Запуск OmegaT из командной строки (все операционные системы)
============================================================

Для запуска OmegaT введите команду:

    cd <путь к папке, в которой находится OmegaT.jar>

    <полный путь с именем к исполняемому файлу Java> -jar OmegaT.jar

(Исполняемый файл Java — это файл java в GNU/Linux или java.exe в Windows.  Если Java установлена на уровне системы и путь к исполняемому файлу добавлен к переменной PATH, то при запуске OmegaT полный путь к Java вводить не обязательно.)
